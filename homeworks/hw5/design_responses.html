Reading Questions

  • What is something you learned from this article? (3-4 sentences)

    Designs utilize a lot of animated features to help guide the flow of the user interactions. It makes transitions between different aspects add more character and flowing transitions to the design rather than causing abrupt changes. For example, in "Selected Works Transitions Experiements" by Zhenya Rynzhuk, the use of animated features helps create the transitions between viewing the different pictures interesting while eliminating any hardcuts that ruin the flow of the design. These features also just add to the characteristic of the design to emphasize the design's/product's purpose. For example, "Prism" by Ruslan Siiz utilizes a scrolling feature where additional items pop up as the user continues to scroll, and even includes moving backgrounds to really highlight what they do.
