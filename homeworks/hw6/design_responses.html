On a smaller screen, the duration of an animation should be shorter because there is less distance whereas a larger screen should have a longer animation duration because there is a larger screen. This is because the animation cannot be too slow or too fast in order to appeal to the user. Longer duration on a small screen would be too slow, and a fast duration on a large screen would be too fast. It's because we expect the web to be fast as it is connected to the internet and should yield instant reponses, so having an animation with a longer duration would not be effective because the user might think that the computer is slow to load or the internet is not working.
